# Jano App Backend

A RESTful API developed with NodeJS and ExpressJS for manipulating the the firebase project of Jano App

## Installation

Use the package manager [npm](https://nodejs.org/en/download/) to install foobar.

```bash
npm i --save
```

## Starting the Server

```bash
npm start
```
